### Hi, I'm Stefan.

I make random things for the love of the game :)

Currently tinkering with **[JSPP](https://github.com/StefanDjurkic/jspp)** (a TypeScript-flavored language that compiles to modern C++20) and **[ZeroEngine](https://github.com/StefanDjurkic/zeroengine)** (a Rust game engine that ships to WebAssembly).

You can see both of them working together in the browser:

**[stefandjurkic.github.io/zeroengine](https://stefandjurkic.github.io/zeroengine/)** - a live demo where JSPP drives a real 3D cube in Three.js, plus a playground with 2D and 3D modes to write and run JSPP code yourself.
